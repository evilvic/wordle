**Wordle** is a web-based word game created and developed by Welsh software engineer Josh Wardle, and owned and published by The New York Times Company since 2022.

# Versions

```
node --version
v16.15.1

npm --version
8.12.1
```

The app was created with *Vite*.
```
npm create vite@latest wordle -- --template react
```

# Installation

To install **Wordle**, first you need to clone this git repository.
```
git clone git@github.com:evilvic/wordle.git
```

Then:

``` 
cd wordle
npm install
npm run dev
```
